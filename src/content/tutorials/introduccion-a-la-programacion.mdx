¡Bienvenido al mundo de la programación! Si estás leyendo esto, es porque has decidido dar el paso de aprender a programar. Y créeme: es una de las mejores decisiones que podrías tomar en 2026.

## ¿Qué es programar?

**Programar es el arte de dar instrucciones precisas a un ordenador para que realice tareas específicas.** Es como escribir una receta de cocina, pero en lugar de preparar un pastel, estás diciéndole al ordenador qué hacer paso a paso.

Piensa en tu día a día: cuando usas Instagram, cuando buscas algo en Google, cuando pides comida a domicilio, cuando ves una serie en Netflix... detrás de cada una de esas acciones hay **miles de líneas de código** escritas por programadores como tú (o como serás pronto).

La programación no es magia, ni es "solo para genios". **Es una habilidad que se aprende con práctica y paciencia**, como tocar un instrumento o hablar un nuevo idioma.

## ¿Cómo ejecutan código los ordenadores?

Los ordenadores solo entienden un lenguaje: **código binario** (ceros y unos). Pero escribir directamente en binario sería una locura. Por eso existen los **lenguajes de programación**: nos permiten escribir instrucciones de forma comprensible para los humanos, y luego esas instrucciones se traducen a código máquina.

El proceso simplificado es:

1. **Tú escribes código** en un lenguaje de programación (como Python)
2. Un **intérprete o compilador** traduce ese código a lenguaje máquina
3. El **procesador ejecuta** las instrucciones

Python es un lenguaje **interpretado**, lo que significa que el código se traduce y ejecuta línea por línea en tiempo real. Esto lo hace perfecto para aprender: puedes ver resultados inmediatos.

## ¿Qué son los lenguajes de programación?

Un **lenguaje de programación** es un conjunto de reglas y sintaxis que nos permite comunicarnos con el ordenador. Igual que en español tenemos gramática y vocabulario, en programación tenemos estructuras y palabras clave.

Existen cientos de lenguajes de programación, cada uno diseñado para diferentes propósitos:

- **Python**: Desarrollo web, inteligencia artificial, automatización, ciencia de datos
- **JavaScript**: Desarrollo web (frontend y backend)
- **Java**: Aplicaciones empresariales, Android
- **C/C++**: Sistemas operativos, videojuegos de alto rendimiento
- **Rust**: Sistemas de bajo nivel con alta seguridad
- **Go**: Servicios de backend escalables

Cada lenguaje tiene sus fortalezas, pero hoy en día, **Python es el lenguaje más demandado del mercado** para la mayoría de aplicaciones modernas.

## ¿Por qué Python?

Python no es solo un excelente lenguaje para principiantes. **Es el lenguaje que usan Google, Netflix, Instagram, Spotify, NASA y miles de empresas más** en producción. No estás aprendiendo un lenguaje "de juguete": estás aprendiendo la misma herramienta que usan los profesionales.

### Ventajas de Python para empezar:

**1. Sintaxis clara y legible**

Python está diseñado para que el código se parezca al lenguaje humano. Compara:

```python
if age >= 18:
    print("Eres mayor de edad")
```

Versus otros lenguajes más verbosos:

```java
if (age >= 18) {
    System.out.println("Eres mayor de edad");
}
```

**2. No necesitas preocuparte por detalles técnicos al principio**

Python gestiona automáticamente la memoria, los tipos de datos y muchos otros aspectos que en otros lenguajes tendrías que controlar manualmente.

**3. Ecosistema gigantesco**

Python tiene **más de 400.000 librerías** disponibles para casi cualquier tarea: desde crear páginas web hasta entrenar modelos de inteligencia artificial.

**4. Demanda laboral altísima**

En 2026, Python es **el lenguaje más buscado por las empresas** según Stack Overflow, GitHub y LinkedIn. Aprenderlo te abre puertas.

**5. Se puede usar profesionalmente**

A diferencia de lenguajes puramente educativos, Python es usado en producción por:

- **Google** (YouTube, buscador)
- **Instagram** (backend completo)
- **Netflix** (sistemas de recomendación)
- **Spotify** (análisis de datos)
- **Dropbox** (servidor y cliente)

### ¿Qué puedes hacer con Python?

La pregunta debería ser: **¿qué NO puedes hacer con Python?**

- **Desarrollo web**: Django, Flask, FastAPI
- **Inteligencia Artificial y Machine Learning**: TensorFlow, PyTorch, scikit-learn
- **Ciencia de datos**: Pandas, NumPy, Matplotlib
- **Automatización**: Scripts para tareas repetitivas
- **Testing y DevOps**: Pytest, Ansible
- **Videojuegos**: Pygame
- **Desktop apps**: PyQt, Tkinter
- **Ciberseguridad**: Hacking ético y pentesting

## Lo que aprenderás en este curso

Este no es un curso cualquiera de Python. **Vas a aprender Python como se escribe en 2026 en empresas profesionales**, no Python "básico" que luego tendrás que desaprender.

A partir de la lección 17, cuando ya tengas fundamentos sólidos, introduciremos:

- **Tipado estático con mypy**: Como se hace en producción
- **Testing con pytest**: TDD desde el principio
- **Formateo automático**: black, isort, flake8
- **Gestión de versiones**: asdf para múltiples versiones de Python
- **Buenas prácticas**: Código limpio y mantenible

No te preocupes si estos términos no te suenan ahora. **Iremos paso a paso**, empezando desde cero absoluto.

## ¿Qué necesitas para empezar?

Solo tres cosas:

1. **Un ordenador** (Windows, Mac o Linux, da igual)
2. **Conexión a internet** (para descargar Python e instalar herramientas)
3. **Ganas de aprender** (lo más importante)

No necesitas:

- ❌ Ser un genio de las matemáticas
- ❌ Tener conocimientos previos de programación
- ❌ Un ordenador potente (Python funciona incluso en ordenadores antiguos)
- ❌ Pagar nada (Python es gratis, este curso es gratis)
