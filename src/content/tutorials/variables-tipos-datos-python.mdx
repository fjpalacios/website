## ¬øQu√© es una variable?

Una variable es un **nombre que le das a un dato** para poder usarlo m√°s adelante. Piensa en ella como una caja etiquetada donde guardas algo.


Acabas de crear tres variables:

- `edad` contiene el n√∫mero `25`
- `puntuacion` contiene el n√∫mero `1500`

### Crear una variable

En Python, crear una variable es tan simple como escribir un nombre, el signo `=`, y el valor que quieres guardar:

```python

```

No hace falta declarar el tipo de dato (como en otros lenguajes). Python es lo suficientemente inteligente para saber que `"Hola, mundo"` es un texto.

### Usar una variable

Una vez creada, puedes usar la variable escribiendo su nombre:


Tambi√©n puedes usar variables dentro de operaciones:

### Las variables pueden cambiar

De ah√≠ viene su nombre: son _variables_. Puedes cambiar su valor en cualquier momento:

## Reglas para nombrar variables

Python tiene algunas reglas estrictas (y otras que son buenas pr√°cticas):

### Reglas obligatorias

1. **Solo letras, n√∫meros y guiones bajos**: No puedes usar espacios ni s√≠mbolos especiales


2. **Nombres descriptivos**: que se entienda qu√© contiene la variable

   ```python
   edad_usuario = 25           # ‚úÖ Tambi√©n v√°lido
   ```

## Tipos de datos b√°sicos

Cuando creas una variable, Python le asigna autom√°ticamente un **tipo de dato** dependiendo del valor que le das. Los tres tipos m√°s importantes para empezar son:

### 1. N√∫meros enteros (int)

Los n√∫meros sin decimales. Se usan para contar, √≠ndices, edades, puntuaciones...

```python

jugadores = 4

```

Puedes hacer operaciones matem√°ticas con ellos:

```python

division = 10 / 5    # 2.0 (ojo: el resultado es float)

resto = 10 % 3       # 1 (resto de la divisi√≥n)
potencia = 2 ** 3    # 8 (2 elevado a 3)
```

### 2. N√∫meros decimales (float)

Los n√∫meros con decimales. Se usan para mediciones, precios, porcentajes...

```python
precio = 19.99
temperatura = 36.5
porcentaje = 0.15
```

**Importante**: En Python se usa el **punto** (`.`) como separador decimal, no la coma:

```python

```

Operaciones con floats:

```python

mitad = 10.0 / 2     # 5.0
```

Cuando mezclas `int` y `float`, el resultado siempre es `float`:

```python
resultado = 10 + 2.5    # 12.5 (float)
```

### 3. Cadenas de texto (str)

Cualquier texto va entre comillas. Puedes usar comillas simples `'...'` o dobles `"..."`, da igual:

```python

```

**Consejo**: Usa comillas dobles por defecto. Solo usa simples cuando necesites comillas dobles dentro del texto:

```python

```

#### Concatenar strings

Puedes unir textos con el operador `+`:


#### f-strings: la forma moderna de formatear texto

Desde Python 3.6 existe una forma mucho m√°s c√≥moda de incluir variables dentro de un texto: los **f-strings** (formatted strings). Pon una `f` antes de las comillas y escribe las variables dentro de `{}`:

```python

# Forma antigua (funciona pero es fea)

# Forma moderna con f-strings (¬°mucho mejor!)

```

Puedes hacer operaciones dentro de las llaves:

```python

descuento = 20
print(f"Precio final: {precio - descuento} euros")  # Precio final: 80 euros
```

### 4. Booleanos (bool)

Solo pueden tener dos valores: `True` (verdadero) o `False` (falso). Se usan para condiciones, decisiones, estados...

```python

```

**Importante**: `True` y `False` van con la primera letra en **may√∫scula**. Es obligatorio:

```python

```

Los booleanos suelen venir de comparaciones:

```python

```

## Convertir entre tipos

A veces necesitas convertir un tipo de dato a otro. Python tiene funciones para eso:

### str() ‚Äî Convertir a texto

```python

print("Tengo " + edad_texto + " a√±os")  # Tengo 25 a√±os
```

O mejor con f-strings (que hace la conversi√≥n autom√°ticamente):

```python

```

### int() ‚Äî Convertir a entero

```python

```

‚ö†Ô∏è **Cuidado**: Si intentas convertir algo que no es un n√∫mero, Python lanzar√° un error:

```python

```

### float() ‚Äî Convertir a decimal

```python

precio = float(precio_texto)
print(precio)  # 19.99
```

### bool() ‚Äî Convertir a booleano

```python

print(es_verdadero)  # True
```

**Regla importante**: En Python, `0`, `""` (string vac√≠o), `None`, listas vac√≠as `[]` y diccionarios vac√≠os `{}` se consideran `False`. Todo lo dem√°s es `True`:

```python

```
